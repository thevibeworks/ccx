* Architecture

#+BEGIN_SRC
ccx/
  cmd/ccx/main.go        Entry point
  internal/
    cmd/                  Cobra commands
    parser/               JSONL parsing + tree building
    render/               Output formats
    web/                  HTTP server + templates
    db/                   SQLite persistence
    config/               Configuration paths
#+END_SRC

Key insight: Claude Code sessions are *trees*, not flat lists.
Uses parentUuid, isCompactSummary, isSidechain for structure.
